Padrões de Design DP-Aut: Um Conjunto de Design Patterns Específicos para o Design de Interfaces para Usuários Autistas

Motivações

O Transtorno do Espectro Autista (TEA) é um transtorno do neurodesenvolvimento que afeta a interação social, comunicação e padrões de comportamento. Tornar a Web mais acessível a todos é um desafio, considerando a pluralidade e diversidade de seus usuários, requerendo um conjunto de soluções complexas em termo de design e desenvolvimento que abranja o espectro autista.

Visando promover a acessibilidade dentro do contexto de Interação Humano-Computador, é proposto então um conjunto de padrões de design para o design de interfaces de aplicações voltadas especificamente ao público autista. Para isso, identificamos na literatura os atributos de qualidade mais importantes e exemplos de aplicações direcionadas a usuários com TEA, culminando no desenvolvimento do conjunto de Padrões de Design "Design Patterns DP-Aut".
